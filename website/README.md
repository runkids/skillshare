# Website

Documentation site for `skillshare`, built with [Docusaurus](https://docusaurus.io/).

## Install

```bash
npm install
```

## Local Development

```bash
npm start
```

## Quality Checks

```bash
npm run typecheck
npm run clear
```

## Build and Preview

```bash
npm run build
npm run serve
```

## Deployment

```bash
npm run deploy
```

If you are using GitHub pages for hosting, this command is a convenient way to build the website and push to the `gh-pages` branch.
